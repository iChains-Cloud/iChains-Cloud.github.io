# 生活脚印

不同世俗的耳语，只看自己喜欢的风景。

# 精选

在南大的图书馆下面的书店拍的，还盖了纪念章。

微风轻舞着落叶，

云朵在湖面摇曳，

感谢秋天，

赠予我的一川风月。

读书的意义大概就是：

用生活所感去读书，

用读书所得去生活。

你知道，有些鸟儿是注定不会被关在牢笼里的，他们的每一片羽毛都闪耀着自由的光辉。

——《肖生克救赎》

You know some birds are not meant to be caged .Their features are just too bright.

——（The Shawshank Redemption，1994）

# 杭州 -- 烟雨江南美如画

总要去趟杭州吧，去吹吹西湖的风，去九溪烟树看看水，去眺望雷峰塔，去断桥看看来来往往的人。感受藏在风里的自由。

当时正好是亚运会的时候，杭州的人超级多，去西湖看灯秀人挤人，不过灯光秀确实很好看很精彩！

买了一些亚运会的吉祥物

去了东方之门，那也得去看看杭州之门

琮琮 宸宸 莲莲

去的一个杭州美食街

馋死我了

晚上，从这里看洪崖洞真的很美！

# 苏州

浓浓的江南风情！

和高中同学一起去的江苏，风景确实很棒，东方之门很壮观！

当时还去了寒山寺，就是姑苏城外寒山寺，夜半钟声到客船的寒山寺

苏州的动物园也去了，虎丘区的美食街，拙政园，苏州中心广场，淮海街等等
